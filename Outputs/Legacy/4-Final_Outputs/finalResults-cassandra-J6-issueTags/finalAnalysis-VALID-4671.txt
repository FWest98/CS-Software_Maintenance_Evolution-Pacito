Description: 

CLI drop index doesn't accept requests to drop columns whose hex names include only
numeric characters.  The 617070 column name below is col2.

<span
class="error">&#91;default@Host&#93;</span> use MyKeyspace;<br/>Authenticated to keyspace:
MyKeyspace<br/><span class="error">&#91;default@Host&#93;</span> drop index on
MyCF.617070; <br/>Syntax error at position 22: mismatched input '617070' expecting
Identifier

While drop index seems to parse correctly with alpha chars included:

<span
class="error">&#91;default@Host&#93;</span> drop index on MyCF.617070x;<br/>Column
'617070x' definition was not found in ColumnFamily 'MyCF'.<br/><span
class="error">&#91;default@Host&#93;</span> drop index on MyCF.col2;<br/>Column 'col2'
definition was not found in ColumnFamily 'MyCF'.
