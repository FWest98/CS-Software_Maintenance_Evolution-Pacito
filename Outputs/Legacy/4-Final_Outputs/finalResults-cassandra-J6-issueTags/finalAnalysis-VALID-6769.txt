Description: 

When issuing a cql select statement with an ORDER BY ... DESC clause the comparison
predicates in the WHERE clause gets reversed. 

Example: (see also attached)

SELECT
number FROM test WHERE number &lt; 3 ORDER BY number DESC

returns the results expected of
WHERE number &gt; 3
