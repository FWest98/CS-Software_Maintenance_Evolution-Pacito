Description: 

The class
org.apache.mina.util.Log4jXmlFormatter
will only output the contents of the
SLF4J MDC if the MDCAdapter used is an
instance of BasicMDCAdapter. This has
two problems:<br/>1) It excludes Log4j
and Logback, the MDCAdapters for which
do not extend BasicMDCAdapter<br/>2) It
creates a dependency on (what should be)
an internal SLF4J class
