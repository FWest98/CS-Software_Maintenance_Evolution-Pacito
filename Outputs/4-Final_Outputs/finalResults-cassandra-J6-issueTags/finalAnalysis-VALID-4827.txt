Description: 

This patch adds support for <tt>BigDecimal</tt> to the <tt>ResultSet</tt> using the
recently added <tt>DecimalType</tt> data type. 

It supports translation from a column
that contained the following Java (CQL) datatypes:
<ul class="alternate"
type="square">	<li><tt>Long - (bigint)</tt></li>	<li><tt>Double -
(double)</tt></li>	<li><tt>BigInteger - (varint)</tt></li>	<li><tt>BigDecimal -
(decimal)</tt></li>	<li><tt>String  - (ascii,text,varchar)</tt></li></ul>
